Охлаждаемый стол для раскатки пиццы — это два в одном: рабочая поверхность для сбора, сервировки блюда и холодильный шкаф, в котором хранятся в оптимальных условиях ингредиенты. У нас можно купить холодильный стол для приготовления пиццы по выгодной цене: в ассортименте — оборудование Atesy, Bolarus, Gastrorag, Hicold, Sagi и других надежных производителей.

Важные параметры

  • Размер столешницы. Для небольшой пиццерии достаточно стола длиной 90-100 см, при большой производительности лучше выбрать модель с большими размерами.
  • Комплектация. Охлаждающий стол имеет полки-решетки с направляющими для хранения заготовок из теста, может комплектоваться дополнительными модулями.
  • Материал столешницы. Каменная (гранитной, мраморной) столешница обеспечивает наиболее высокое качество готового продукта, стальная (из нержавеющей стали) — представляет собой бюджетный и простой в уходе вариант.

Купите стол-холодильник для пиццы в компании «КофеБутик» в Москве — профессиональное оборудование оптимизирует рабочий процесс в вашем заведении: увеличит удобство, эффективность работы и скорость готовки.
